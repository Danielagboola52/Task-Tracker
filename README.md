# Task Tracker

A simple, responsive to-do list app built with HTML, CSS, and JavaScript.

## Features
- Add tasks with Enter key
- Mark as complete (strikethrough + move to bottom)
- Delete tasks
- Dark/Light mode toggle
- Celebration message when all tasks are completed
- Responsive for mobile/desktop

## How to Run
1. Open `index.html` in any browser.
